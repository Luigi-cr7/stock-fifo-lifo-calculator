# Stock FIFO/LIFO Calculator 

A C++ program that calculates stock cost basis using 
both FIFO and LIFO accounting methods.

## How It Works
1. Enter stock purchases with symbol, shares, and price
2. Choose a stock and number of shares to sell
3. Get the average cost basis using both FIFO and LIFO methods

## Concepts Used
- Queue (FIFO) and Stack (LIFO) data structures
- Structs for custom data modeling
- Partial share selling logic
- Menu-driven program design

## What I Learned
- How FIFO and LIFO work in real financial accounting
- How queues and stacks solve real-world problems
- How to implement partial transaction logic in C++
